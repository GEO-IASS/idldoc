/pub/training/IDL_intro/
--------------------------------------------------------------------

This directory contains compressed and self-extracting versions of the
files used in the course "Introduction to IDL." Note that
these files are meant to be used in conjunction with the Introduction
to IDL course manual; the use of these files without the manual is
permitted, but not recommended.

Extract and view the file 'index.html' for further information on the
programs included in these files.
